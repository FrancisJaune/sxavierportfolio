Project 3: Linear Regression Model¶

Introduction¶

For project 3, I have trained a multiple linear regression model to predict the rental price of NYC. Three models were trained for this project. The first, and most simple model only used the most basic and easy to process data to predict price. The second model, in addition to using easier to process data, also used the words within the name of listings, and the third model used PCA to reduce the dimension of the data from the names.

The dataset contains the 16 columns which record data such as the name of the property, the properties rating, and the price to rent for a night.

Regression is the prediction of a continous value using features correlated to the value being predicted. Linear regression looks for a linear relationship between features and the value being predicted. Multiple linear regression can be represented mathmatically as ...

$y = b + m_ 1 x_1 + m_ 2 x_2 + ... + m_n x_n$

for a model with n features

Experiment 2¶

For the second experiment, the features of name were split into a list, and the words that were present inthe names of at least 50 listings. These words were then used to make columns containing binary data on whether or not the word is present in the name of a listing. These columns were used along with the other features also used in experiment 1 to train this model. SInce the training data frame in this experiment contained so many columns, a loop was made to itterate through the p values, and if a p value was greater than 0.2, the feature associated with it was removed. The R-squared value for this model was significantly higher than the R-sqaured of model 1, but was low overall. This still shows correlation between price and the words within the name of a property.

Impact¶

When it comes to short term rentals and expensive cities like NYC, there is the issue of short term rentals raising the already high housing prices by decreasing the housing supply. NYC has had issues with landlords putting apartments that could be rented to tenants on services like AirBnB, where the landlord can easily make hundreds of dollards a day on the room, and models such as the ones built in this project could be used by property owners to find out whether or not their property could get a high price on AirBnB or another short term rental service. This could cause an increase in the cost of housing for those who live in the city.

Conclusion¶

The models built in this project did not have the best accuracy, but did provide good practice for building regression models. With this dataset, though there was some correlation between the features and price, the prices could also be affected by other features of the property that are not shown in the dataset such as the size of the rental. Also, these prices are decided by the property owners. It would be useful to have data on how often the rentals ae booked, as it is possible that some of the properties are overpriced, and this are not rented often.
